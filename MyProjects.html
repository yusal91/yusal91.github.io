Last Fantasy: The End

This project is a third-person action RPG that emphasizes a rich narrative experience. Drawing inspiration from open-world games, it features a teleportation system and an inventory management system that utilizes drag and drop mechanics. Additionally, elements from the Final Fantasy series have influenced the design, particularly the dashing ability, which is heavily inspired by Dissidia Final Fantasy. While combat mechanics are still under development, they are also being shaped by these influences.

This is a display of the content I've created up to now. While it's not perfect yet, I'm continuously working on improving it whenever I have the chance.

This example highlights the content I want to enhance. An inventory system like this should include a category button so users can easily see items sorted by their categories. Plus, I want to incorporate text prompts for interactions, like when a player gets close to a save point, door, or treasure chest, and so on.
